1. Цель проекта
Создать веб-сайт, позволяющий пользователю:
Регистрироваться и авторизовываться.
Загружать, просматривать, скачивать и удалять файлы.
Создавать и управлять папками.
Взаимодействовать с файлами через REST API.
2. Основные функции сайта
2.1. Регистрация и авторизация
Регистрация по email и паролю.
Вход в систему.
Выход из системы.
2.2. Управление файлами
Загрузка файлов (ограничение по типу и размеру).
Просмотр списка файлов.
Скачивание файлов.
Удаление файлов.
2.3. Управление папками
Создание папок.
Просмотр содержимого папок.
Перемещение файлов между папками.
Удаление папок (с файлами).
2.4. Поиск файлов
Поиск по названию файла.
2.5. REST API
Получение списка файлов: GET /api/files
Загрузка файла: POST /api/upload
Удаление файла: DELETE /api/files/<id>
Получение списка папок: GET /api/folders
