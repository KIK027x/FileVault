FileVault — Личный файловый менеджер

Что это такое
Простой и удобный сайт для хранения твоих файлов, похожий на Google Диск или Яндекс.Диск, но написанный полностью на Python. Всё работает у тебя на компьютере или на сервере.

Что умеет проект

🔐 Вход и безопасность
• Можно зарегистрироваться и войти под своим логином/паролем.
• Пароли надёжно шифруются (никто, даже разработчик, не увидит твой пароль).
• Каждый пользователь видит только свои файлы. Чужие файлы недоступны.

📁 Работа с папками
• Можно создавать папки и вкладывать одни папки в другие (как в обычном Проводнике Windows).
• Сайт показывает, сколько весит папка целиком (вместе со всем, что внутри).
• Папку можно удалить вместе со всем содержимым одной кнопкой.
• Целую папку можно скачать одним ZIP-архивом.

📄 Работа с файлами
• Загрузка любых файлов: картинки, документы, видео, музыка, архивы, код.
• Если загрузить файл с таким же именем, он не перезапишет старый, а получит название файл_1, файл_2 и т.д.
• Файлы можно скачивать обратно.
• Текстовые файлы (.txt, .py, .json, .html) можно посмотреть прямо на сайте, не скачивая их.
• Картинки отображаются в красивой отдельной галерее.

🏷️ Теги и поиск
• При загрузке файла можно написать теги через запятую (например: важное, работа, 2026).
• Есть строка поиска: она ищет файлы и по названию, и по тегам.
• На главной странице показывается список последних загруженных файлов.

🛠️ Дополнительные фишки
• Экспорт данных: Можно скачать весь список своих файлов в виде таблицы Excel (CSV) или текстового файла (JSON).
• REST API: Программисты могут управлять файлами через специальные ссылки (загружать, удалять, смотреть список) без захода на сайт.
• Адаптивный дизайн: Сайт удобно смотрится и на компьютере, и на телефоне.

Технологии (на чём сделано)
• Язык: Python
• Веб-фреймворк: Flask
• База данных: SQLite (хранится в одном файле, легко переносить)
• Внешний вид: Bootstrap 5 (красивые кнопки, таблицы, окна)

Как запустить
1. Скачать код.
2. Установить библиотеки командой pip install -r requirements.txt.
3. Запустить файл app.py.
4. Открыть в браузере адрес http://localhost:5000.
